# Project Brief

Dự án **ProG Coder Shop Microservices** là một nền tảng thương mại điện tử hoàn chỉnh được xây dựng trên kiến trúc microservices sử dụng .NET Core 8.

## Mục tiêu chính
- Xây dựng một hệ thống thương mại điện tử có khả năng mở rộng, bảo trì và kiểm thử tốt.
- Áp dụng các mẫu thiết kế hiện đại như Microservices, Clean Architecture, DDD, CQRS, Event-Driven Architecture.
- Cung cấp một bộ demo đầy đủ từ backend đến frontend, kèm theo hạ tầng Docker/Kubernetes.

## Phạm vi dự án
- **Backend**: Microservices cho Catalog, Basket, Order, Inventory, Discount, Notification, Search, Report, và Communication.
- **Frontend**: Ứng dụng quản trị (App.Admin) và giao diện mua sắm (App.Store) sử dụng React.
- **Hạ tầng**: API Gateway (YARP), Message Broker (RabbitMQ), Identity Provider (Keycloak), Monitoring stack (Prometheus, Grafana, Loki, Tempo).
